# 中文GPT训练完整指南

## 🚀 快速开始

### 1. 准备数据

进入中文数据目录：
```bash
cd data/chinese_literature
```

使用示例数据（推荐新手）：
```bash
python prepare_sample.py
```

或使用完整数据（需要网络）：
```bash
python prepare.py
```

### 2. 选择训练方式

#### 方式一：从头训练（GPU）
```bash
python train.py config/train_chinese_literature.py
```

#### 方式二：CPU训练
```bash
python train.py config/train_chinese_literature_cpu.py
```

#### 方式三：微调预训练模型（推荐）
```bash
python train.py config/finetune_chinese_literature.py
```

### 3. 生成中文文本

训练完成后：
```bash
python generate_chinese.py --start="红楼梦第一回" --num_samples=3 --max_new_tokens=200
```

## 📊 配置说明

### GPU配置 (train_chinese_literature.py)
- 模型大小：6层，384维
- 批处理：64
- 序列长度：256字符
- 预计时间：30-60分钟

### CPU配置 (train_chinese_literature_cpu.py)
- 模型大小：4层，128维
- 批处理：12
- 序列长度：64字符
- 预计时间：2-4小时

### 微调配置 (finetune_chinese_literature.py)
- 基于GPT-2
- 学习率：5e-5（较小）
- 迭代次数：1000
- 预计时间：10-30分钟

## 🎯 生成参数调节

- `--temperature`：控制随机性（0.1-1.5）
  - 0.1：非常确定性的文本
  - 0.8：平衡的创造性（推荐）
  - 1.5：非常有创意但可能不连贯

- `--top_k`：只从前k个候选词中选择
  - 1：贪婪搜索
  - 40：适中选择
  - 200：更多样性

- `--max_new_tokens`：生成文本长度
  - 50：短句
  - 200：段落
  - 500：长文章

## 🛠️ 常见问题

### Q: CUDA内存不足怎么办？
A: 减少batch_size和block_size：
```bash
python train.py config/train_chinese_literature.py --batch_size=32 --block_size=128
```

### Q: 生成的文本质量不好？
A: 尝试以下方法：
1. 增加训练时间（max_iters）
2. 使用微调而非从头训练
3. 调整temperature参数
4. 增加训练数据

### Q: 只有CPU如何加速？
A: 
1. 减小模型参数
2. 设置compile=False
3. 使用最新PyTorch版本

### Q: 如何添加自己的文本？
A: 将txt文件放在data/chinese_literature/目录下，修改prepare.py即可。

## 📈 进阶使用

### 自定义数据集
1. 将你的中文文本文件（.txt）放在`data/chinese_literature/`
2. 修改`prepare.py`中的文件路径
3. 重新运行数据准备

### 模型调优
修改配置文件中的参数：
- `n_layer`：增加层数提高能力但增加计算量
- `n_embd`：增加嵌入维度
- `learning_rate`：学习率调节
- `dropout`：防过拟合

### 多GPU训练
```bash
torchrun --standalone --nproc_per_node=2 train.py config/train_chinese_literature.py
```
